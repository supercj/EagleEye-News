## 二、商店描述

### 简短描述（Chrome ≤132 字符 / Edge ≤256 字符）

> 无服务器的浏览器资讯聚合扩展，内置 36+ 公开源，支持自定义订阅、收藏、已读和自动刷新，数据完全保存在本地。

### 详细描述（Chrome Web Store）

```
🦅 EagleEye News — 鹰眼资讯

一个开源、无服务器、数据完全本地的浏览器资讯聚合扩展。无需注册账号，一个弹窗掌握全网热点。

【核心功能】

📰 多源聚合
内置 36+ 公开资讯源，涵盖综合新闻、中文热门、科技、开发者、AI、财经六大分类。包括 36氪、IT之家、少数派、Hacker News、GitHub Trending、The Verge、TechCrunch 等知名站点。

📥 自定义订阅
支持导入 OPML 文件、RSS / Atom / JSON Feed 地址，也支持逐行粘贴订阅链接。轻松接入你关注的任何公开源。

🔍 智能筛选
按标签分组浏览，支持关键词搜索标题、来源、摘要和域名。一键「选择本类 / 取消本类」，快速管理订阅。

⭐ 收藏 & 已读
本地记录收藏和已读状态，支持「隐藏已读」过滤模式，让你专注于未读内容。

🔄 自动刷新
后台定时抓取最新资讯，支持 1 分钟到 4 小时的自定义刷新间隔。也支持手动一键刷新。

⌨️ 快捷键
Ctrl+Shift+E（Mac: Cmd+Shift+E）一键打开弹窗。

🌙 深色模式
跟随系统主题自动切换，护眼舒适。

🔒 隐私优先
所有数据仅存储在浏览器本地（chrome.storage.local），不采集、不上传、不追踪。卸载即清除。

【数据合规】
只聚合公开来源中的标题、链接、来源、发布时间和短摘要，不复制全文，不绕过登录或付费墙。

开源地址：https://github.com/supercj/EagleEye-News
隐私说明：https://github.com/supercj/EagleEye-News/blob/main/PRIVACY.md
```

## 七、权限说明（审核必备）

审核人员会检查权限合理性，以下是每个权限的用途说明：

| 权限 | manifest 中的声明 | 审核说明 |
| --- | --- | --- |
| `storage` | 保存用户设置、订阅源、收藏和已读状态 | 核心功能必需 |
| `alarms` | 后台定时刷新资讯 | 核心功能必需 |
| `permissions` | 导入自定义订阅源时请求站点权限 | 用户主动触发 |
| 主机权限 | 访问内置资讯源和用户导入的源 | 仅用于获取公开内容 |

> 主机权限使用 `<all_urls>` 作为 optional_host_permissions，仅在用户导入自定义源时按需申请。

## 九、版本更新流程

1. 修改 `manifest.json` 中的 `version` 字段
2. 更新 `CHANGELOG.md`（如已创建）
3. 重新打包 `.zip`
4. 分别在 Chrome Web Store 和 Edge Add-ons 上传新版本
5. 填写更新说明
6. 提交审核
