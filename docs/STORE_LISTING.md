# 上架素材清单

## 基本信息

- 名称：EagleEye News
- 中文名：鹰眼资讯
- 简短描述：无服务器的 Chrome/Edge 热点资讯聚合扩展，支持导入订阅源，并在本地保存收藏与已读状态。
- 类别建议：新闻与天气 / 生产力工具
- 语言：中文（简体）

## 长描述草稿

EagleEye News 是一个本地运行的资讯聚合扩展，用于把公开 RSS、Atom、JSON Feed、公开 API 和榜单页面中的热门链接聚合到浏览器弹窗中。

扩展支持按订阅来源筛选、关键词搜索、来源开关、按标签分组的推荐源排名、本地收藏、已读状态记录和定时刷新。用户也可以导入 OPML 文件或逐行粘贴订阅源地址，扩展会在获得浏览器权限后把订阅源保存到本地。

EagleEye News 不需要账号，不依赖自有服务器，不采集个人信息。文章、收藏、已读状态和用户导入的订阅源都保存在浏览器本地。
