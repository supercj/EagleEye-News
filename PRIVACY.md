# EagleEye News 隐私说明

EagleEye News 是一个本地运行的 Chrome/Edge 资讯聚合扩展。扩展不提供账号系统，不采集、上传或出售任何个人信息。

## 数据保存

扩展会在浏览器本地的 `chrome.storage.local` 中保存以下数据：

- 已启用的资讯来源和刷新间隔设置
- 用户导入的 RSS、Atom、JSON Feed 或 OPML 订阅源地址
- 收藏的文章 ID
- 已读的文章 ID
- 最近一次刷新时间和来源刷新状态
- 聚合到的文章标题、链接、来源、发布时间和短摘要

这些数据仅保存在用户自己的浏览器中。卸载扩展或清除浏览器扩展数据后，本地数据会被删除。

## 网络请求

扩展会向用户启用的公开资讯来源发起请求，用于获取 RSS、Atom、JSON Feed、公开 API 或榜单页面内容。导入自定义订阅源时，扩展会向对应源地址请求内容，以识别订阅名称和类型。

扩展不会绕过登录、付费墙、反爬机制或访问限制，也不会复制来源网站的全文内容。

## 权限用途

- `storage`：保存设置、订阅源、收藏、已读状态和文章列表。
- `alarms`：按用户设置的间隔在后台刷新资讯。
- `permissions`：在用户导入自定义订阅源时，请求访问对应站点的权限。
- 主机权限：访问内置资讯源和用户明确导入的订阅源。

## 第三方服务

扩展当前不接入分析、广告、追踪或 AI 服务。若未来新增 AI 摘要等能力，应优先让用户自行提供 API Key，或在新增服务端方案前更新本隐私说明。
